Quick Setup Guide

1
Create a Meta Business Account

Go to business.facebook.com and create or select your business account.

2
Create a WhatsApp Business App

Visit developers.facebook.com/apps → Create App → Select "Business" → Add WhatsApp product.

3
Generate Permanent Access Token

In the App Dashboard → WhatsApp → API Setup → Generate a permanent token with whatsapp_business_messaging permission.

4
Copy Your Phone Number ID

From WhatsApp → API Setup, copy the Phone Number ID and WhatsApp Business Account ID.

5
Configure Webhook

Set the webhook URL below in your Meta App Dashboard and subscribe to messages events.
